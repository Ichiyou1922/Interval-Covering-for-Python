# incovコマンド
![test](https://github.com/Ichiyou1922/Interval-covering-for-Python/actions/workflows/test.yml/badge.svg)

## プログラムの内容
- 標準入力から入力された幾つかの数値を最小個数のクラスタで包み，その個数を標準出力から出力するプログラム．
- クラスタの定義: 中心値Cと半径Rを持ち， $[C-R, C+R]$ の範囲の数値を収めた集合．

## アルゴリズムの概要
このプログラムは，**貪欲法 (Greedy Algorithm)** を用いて最小のクラスタ数を計算します．

1. **ソート**: 入力された数値を昇順にソートします．
2. **貪欲的選択**:
   - まだカバーされていない最小の数値(最も左にある点)に着目します．
   - その数値を左端とする区間 $[x, x+2R]$ を設定します(これにより，その数値をカバーしつつ，可能な限り右側の数値までカバーできます)．
   - この区間に含まれるすべての数値をカバー済みとして扱います．
3. **繰り返し**: すべての数値がカバーされるまで，上記の手順を繰り返します．

この手法により，計算量 $O(N \log N)$(ソートにかかる時間)で最適解を得ることができます．

## 仕様詳細
- **入力**: 標準入力から数値を読み込みます(各行に1つの数値)．
- **出力**: すべての数値をカバーするために必要な最小のクラスタ数．
- **パラメータ**: 現在のバージョンでは，クラスタの半径 $R$ は $1.0$ に固定されています．
- **エラー処理**: 数値として解釈できない入力が含まれる場合，プログラムは終了コード1で停止します．

## 使用方法
```bash
$ git clone git@github.com:Ichiyou1922/Interval-Covering-for-Python.git
$ cd Interval-Covering-for-Python
$ chmod +x incov
$ seq 5 | ./incov #半径R=1の場合seq 5で生成された数値はリスト化され{1, 2, 3, 4, 5}となり{1, 2, 3}, {4, 5}としてクラスタに包まれ，その個数は2である．
2
```
